Maquetación de un Pagina Web

***** Descripción *****

Este proyecto tiene como objetivo mejorar y optimizar el sistema de estilos de un sitio web, Se aplico el uso de *Sass*. Dado que permite escribir un CSS de una manera más eficiente y mantenible, lo que facilita la creación de estilos complejos con menos esfuerzo y mayor organización.

***** Estructura del Proyecto*****

La estructura de carpetas del proyecto está diseñada para mantener el código limpio y bien organizado:

- **dist**: Carpeta destinada a almacenar los archivos de distribución final del proyecto.
- **img**: Carpeta que contiene todas las imágenes utilizadas en el proyecto.
- **node_modules**: Contiene los módulos de Node.js necesarios para el proyecto.
- **scripts**: Carpeta que almacena los archivos de scripts del proyecto.
- **styles**: Carpeta que contiene los archivos de estilos en formato SCSS.

  Dentro de `styles` se encuentran los siguientes archivos:
  - `base.scss`
  - `footer.scss`
  - `header.scss`
  - `main.scss`
  - `mixins.scss`
  - `sections.scss`
  - `variables.scss`

- **index.html**: Archivo HTML principal del proyecto.

****** Funcionalidades ******

- **Optimización del CSS**: Utilización de variables, mixins, y funciones para un código CSS más limpio y reutilizable.
- **Despliegue en Netlify**: La aplicación ha sido desplegada usando Netlify, garantizando un hosting eficiente y gratuito.
- **Responsividad**: El sitio web es completamente responsive.
